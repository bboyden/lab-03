Lab 3: Modern JavaScript and Modules

This lab focused on improving JavaScript code using modern syntax and organizing logic into reusable modules.

Key Skills Learned:

  • Refactored code using arrow functions and object destructuring
  • Used for..in for objects and for..of for arrays
  • Worked with the spread operator
  • Created and imported Node.js modules using module.exports and require()

Lab 3: Making JavaScript Way Cooler

In this lab, I took some older JavaScript code and gave it like a total glow-up. I refactored stuff using arrow functions, destructuring, and all that modern syntax goodness. I also got to mess with for..in and for..of loops, which honestly helped me understand objects and arrays way better. Then I made a module and exported it with module.exports and used require to bring it back in, which was pretty satisfying.
